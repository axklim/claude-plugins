# Second brain

The Layer-3 library of a personal memory system. Sessions are captured to `raw/`, queued in
`inbox/`, and filed here into two indexes.

- **[[wiki/index|Wiki]]** — topical index: what you know about X. Navigate by topic `index.md`
  files and note `summary:` frontmatter.
- **Journal** — temporal index: when/what you did. See `journal/<YYYY-MM>/<YYYY-MM-DD>.md`.
